1. На вход томографичексие снимки (макротомография). Был сделана КТ в медицинском томографе, снимают, поворачивают, получают изображение в виде круга. 
2. Необходимо получать различные данные из макротомографии. Получаем яркость, которую по некоторой зависимости (эмпирически) можно перевести в плотность, а также пересчитывать эффективный атомный номер. 
3. Зная значения плотностей от глубины, можно пересчитатть по формуле Гарднера значения скоростей продольной волны от глубины, из чего можно пересчиитать скорость поперечной волны, как корень из 2. А далее по формулам произвести расчет акустических параметров таких как прочность на одноосное сжатие, модуль Юнга, модуль сжатия, коэффициент Пуассона. 
4. Получим синтетическую сейсмограмму. Для этого необходима характеристика среды и сигнатру источника. Характеристику среды можем пересчитать, зная скорости и плотности, а сигнатуру источника смоделируем, возьмем имульс Риккера. Выполнив свертку данных двух параметров, получим синтетическую сейсмограмму.
5. Можно попробовать сравнить с данными АК, ГК.

beam harding
1. Попытка борьбы с эффектом beam harding (засвечивания изображения на краях). В результате томографии на краях изображения появляются засветы, образующиеся от того, что по краям проходит большее количество фотонов, то есть происходит ослабление рентгеновского пучка в разной степени, а на краях образца в меньшей, то есть проходит большее количество фотонов. Эффект аналогичен фильтру высоких частот. Так как в пучке остаются только фотоны с более высокой энергией, и, таким образом, средняя энергия луча увеливается ("упрочняется").
2. Для начала необходимо выделять область изучения на изображении. То есть выделять окружность, есть какой-то метод Каана, который изет окружности (проход окружностями по изображению). Отличие по цвету, чтобы найти границу изображения (можно брать производную между двумя точками).
3. Построить гистограмму средней яркости от радиуса.
